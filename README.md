
# 🐱 Un Día en la Vida de Tita

## Descripción

Este proyecto es una historia interactiva en HTML titulada **“Un Día en la Vida de Tita”**.

El lector asume el rol de Tita, una gata curiosa que despierta en casa y debe tomar decisiones que determinan cómo se desarrolla su día.

Cada página representa una escena distinta, y las decisiones se realizan mediante enlaces que llevan a diferentes rutas narrativas.

La historia no es lineal y contiene múltiples finales.

---

## Objetivo del Proyecto

Este trabajo fue realizado para practicar:

* Creación de múltiples páginas HTML
* Uso correcto de paths relativos (`./`, `../`, `../../`)
* Uso correcto de paths absolutos para imágenes (`/images/...`)
* Organización estructurada de carpetas
* Uso de etiquetas semánticas de HTML
* Comprensión de cómo funciona la navegación web sin CSS ni JavaScript

---

## Estructura del Proyecto

```
index.html
README.md

/house
  cocina.html
  ventana.html
  siesta.html

/garden
  jardin.html
  arbol.html

/roof
  techo.html
  paloma.html

/images
  gato.png
  techo.jpg
  jardin.jpg
```

* `index.html` es el inicio de la historia.
* Las escenas están distribuidas en carpetas.
* Todas las imágenes están almacenadas en la carpeta `/images`.
* No se utilizaron archivos innecesarios.

---

## Características Técnicas

El proyecto cumple con los siguientes requisitos:

* Uso exclusivo de HTML (sin CSS ni JavaScript)
* Uso correcto de:

  * `<main>`
  * `<article>`
  * `<section>`
  * `<aside>`
  * `<header>` y `<footer>`
  * `<nav>`, `<ul>`, `<li>`
  * `<figure>`, `<img>`, `<figcaption>`
  * `<details>` y `<summary>`
  * `<dialog>` y `<form>`
  * `<progress>`
  * `<time>`
* Implementación de breadcrumbs en todas las páginas
* Múltiples finales
* Al menos un loop de navegación utilizando `../`

---

## Temática

La historia sigue un día en la vida de Tita, una gata que explora:

* La cocina
* La ventana
* El jardín
* El techo
* Un encuentro con una paloma

Las decisiones del lector determinan si Tita se convierte en exploradora valiente, cazadora frustrada o simplemente disfruta de una siesta perfecta.
